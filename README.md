# Cinema Management System

A Java-based system for managing cinema screenings, seat reservations, and ticket sales.

## Technologies
- Java 11+
- Object-Oriented Programming

## Features
- Movie and screening schedule management
- Seat reservation and availability
- Ticket sales and pricing
- Customer records
- Sales report generation

## Getting Started
```bash
javac Main.java
java Main
```
